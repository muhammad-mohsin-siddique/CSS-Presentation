  1. Make A Presentation On The Following Topics On CSS:

    • What Does it mean by cascading in css:

      • The “cascading” in CSS refers to the fact that styling rules “cascade” down from several sources. This means that CSS has an inherent hierarchy and styles of a higher precedence will overwrite rules of a lower precedence.

    • Why We Use CSS

      • We use CSS (Cascading Style Sheets) in web development to control how web pages look and feel. It helps separate design from content, makes websites look consistent, adapts to different devices, and makes it easier to update and maintain the appearance of a website. In simple terms, CSS makes websites pretty and functional.

    • What Does It Mean By Stylesheet In CSS

      • In CSS, a "stylesheet" is a file or document that contains rules and instructions for how a web page's elements should be styled or formatted. Stylesheets define things like colors, fonts, spacing, and layout for HTML elements. By using stylesheets, you can control the visual presentation of your web content, making it look the way you want it to on a web page. Stylesheets can be linked to HTML documents to apply these styles consistently across multiple pages, making it easier to manage and maintain the design of a website.

    • How Many Ways You Can Add Css To HTML

      2. Internal Stylesheet:

        3. Embed CSS directly within the HTML file using <style> in the <head>

      3. Inline Styles::

        4. Apply styles directly to individual HTML elements using the style attribute.
